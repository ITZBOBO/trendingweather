# 🌤 trendingweather Weather App

**Live Demo:** [https://trendingweather.netlify.app/](https://trendingweather.netlify.app/)

A simple weather app built with HTML, CSS, and JavaScript using the OpenWeatherMap API.

## Features

- Search weather by city name
- Displays temperature, weather description, wind speed, humidity, visibility, and feels-like temperature
- Clean glassmorphism UI
- Loading state and error handling

## How to Use

1. Clone the repo
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Open `app.js` and replace `YOUR_API_KEY_HERE` with your key
4. Open `index.html` in your browser

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)
